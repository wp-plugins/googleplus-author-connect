== Description ==\
\
Plugin for displaying Author's Google +1 Profile Picture in Google Search Results.\
\
\
== Installation ==\
\
1. Download the zip file.\
2. Go to Wordpress Admin Panel -> Plugins->Add New->Upload and upload the zip file.\
3. Once upload is complete, activate the plugin.\
4. Go to Settings->GooglePlus Author Connect and fill in the setting of the plugin.\
\
